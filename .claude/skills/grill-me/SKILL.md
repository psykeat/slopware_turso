# Skill: Grill Me

## Goal

Reach a shared understanding of a plan or design by relentlessly interviewing the user and resolving decision tree branches.

## Triggers

- User wants to stress-test a plan.
- User asks to be "grilled" on a design.
- User says "grill me".

## Core Instructions

- **Interview:** Ask questions ONE at a time.
- **Walk the Tree:** Follow decision branches to their conclusion.
- **Recommend:** Always provide a recommended answer for each question you ask.
- **Research First:** If a question can be answered by exploring the codebase, do that instead of asking the user.
- **Goal-Oriented:** Stop when the plan is robust and all major branches are resolved.
